# Official Fenix ecosystem change log

## Version symbols

- f: failed version
- t: testing version
- A,B,C,D...: Subversions, usually added in the same day

> ### Examples of versions
> - 0.7A
> - 0.7Df
> - 0.7
> - 0.7Dt
